# POS Anita

[![EAS Build](https://github.com/OscarLTC/pos-anita/actions/workflows/eas-build.yml/badge.svg)](https://github.com/OscarLTC/pos-anita/actions/workflows/eas-build.yml)

POS Anita started as a personal project built for a family business — a small store run by Anita. The goal was simple: replace the notebook and the mental math with something fast, reliable, and easy to use from a phone. No overengineering, just a point-of-sale app that works.

It's built with **Expo** and **React Native**, targeting Android for now.

---

### Try it

The latest Android build is available on Expo — you can download the APK or scan the QR code directly from the build page:

**[expo.dev/accounts/oscarltc/projects/pos-anita/builds](https://expo.dev/accounts/oscarltc/projects/pos-anita/builds)**

Every push to `main` triggers a new build automatically via EAS Build + GitHub Actions.

---

### Run locally

```bash
npm install
npx expo start
```
